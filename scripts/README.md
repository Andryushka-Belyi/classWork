# Repository Sync Script (Overwrite Mode)

## 📌 Описание

Скрипт выполняет полную синхронизацию одного репозитория с другим:

1. Обновляет исходный репозиторий (`git pull`)
2. Полностью очищает целевой репозиторий (кроме `.git`)
3. Копирует все файлы из исходного репозитория
4. (Опционально) выполняет `git push`

⚠️ В отличие от rsync, используется **жёсткая перезапись (overwrite mode)**.

---

## ⚙️ Особенности

- Полная очистка целевого репозитория
- Исключение `.git`
- Проверка путей
- Защита от совпадающих директорий
- Подходит для CI/CD сценариев

---

## 🐧 Bash запуск

```bash
chmod +x sync.sh
./sync.sh /path/source /path/target
```

## 🪟 PowerShell 
запуск
```powershell
.\sync.ps1 -SourceRepo "C:\repo1" -TargetRepo "C:\repo2"
```
